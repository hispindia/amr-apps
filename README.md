# AMR Data Entry and Approval

### Installation
```bash
cd [root_dir]
yarn install

cd [root_dir]/approval
yarn install

cd [root_dir]/entry
yarn install
```

### Development
You need to log in to the test server: https://amrtest.icmr.org.in/amrtest

```bash
cd [root_dir]
yarn start

cd [root_dir]/approval
yarn start

cd [root_dir]/entry
yarn start
```

### Building
```bash
cd [root_dir]
yarn build
```

### Versioning
```bash
cd [root_dir]
yarn bump x.x.x
```
