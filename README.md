## Installation

```bash
yarn install
```

## Development

You need to log in to the test server: https://amrtest.icmr.org.in/amr

```bash
yarn start
```

You may have to F5 once.

## Building

```bash
yarn build
```

## Versioning

```bash
yarn bump        # 1.1.1 -> 1.1.2
yarn bump minor  # 1.1.1 -> 1.2.0
yarn bump major  # 1.1.1 -> 2.0.0
```
